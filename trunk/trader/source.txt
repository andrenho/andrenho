The source code is organized like this:

 - the source files in the root are the game engine;
 - the subdirectory UI contains the User Interface engine.

The main window is a object of the class UI (in ui/ui.rb). 

The images are loaded into a hash called UI#image. Each file is loaded into
a image stored in this hash, with a symbol with the filename as key. For 
example, a image 'truck.png' is loaded into UI#image[:truck].

The method UI#redefine! replaces the methods of the UI object, depending the 
screen the player is using. For example, if the user accesses the Market screen,
the method UI#redefine! changes (extends) the UI class with the Market
module (in ui/market.rb).
