\section{Limites}

Limites săo usados para calcular a qual valor $f(x)$ tende quando $x$ se
aproxima de um determinado número. Por exemplo, tomando a segunte funçăo:

\[ f(x) = x^2 \]

Ŕ medida que $x$ se aproxima de $2$, $f(x)$ se aproxima de $4$. Isto é
representado por

\[ \lim _{x \to 2} f(x) = 4 \]

\begin{pspicture}(-3,-1)(1.5,3)
  \psset{xunit=0.5cm,yunit=0.5cm} 
  \psaxes{<->}(0,0)(-3,-1.5)(3,5)
  \psplot[linecolor=blue, linewidth=1.5pt]{-2.2}{2.2}{x dup mul}
  \psdot[linecolor=blue, linewidth=1.5pt](2,4)
\end{pspicture}

\newpage

Os limites calculam o quando o valor \textbf{tende a}, e năo o valor específico.
Assim,

\begin{gather*}
  f(x) = \left\{\begin{array}{l l}
                   x^2 & \text{se } x \neq 2 \\
                   3 & \text{se } x = 2
                 \end{array} \right. \\
  \lim _{x \to 2} f(x) = 4
\end{gather*}

\begin{pspicture}(-3,-1)(1.5,3)
  \psset{xunit=0.5cm,yunit=0.5cm} 
  \psaxes{<->}(0,0)(-3,-1.5)(3,5)
  \psplot[linecolor=blue, linewidth=1.5pt]{-2.2}{2.2}{x dup mul}
  \psdot[linecolor=blue, linewidth=1.5pt, dotstyle=o](2,4)
  \psdot[linecolor=blue, linewidth=1.5pt](2,3)
\end{pspicture}

\newpage

Existem casos em que năo é possível chegar no ponto específico do qual o limite
se refere.

\[ \lim _{x \to -1} \frac{2x+2}{x+1} \]
\[ \frac{2\cdot -1+2}{-1+1} = \frac{0}{0} \]

Neste casos, deve-se tentar simplificar a expressăo.

\[ \lim _{x \to -1} \frac{2x+2}{x+1} = \frac{2(x+1)}{(x+1)} = \frac{2}{1} = 2 \]

Pode-se também montar o gráfico, descobrindo o limite intuitivamente.

\begin{pspicture}(-3,-1)(2.5,2)
  \psset{xunit=0.5cm,yunit=0.5cm} 
  \psaxes{<->}(0,0)(-3,-1.5)(4,3)
  \psplot[linecolor=blue, linewidth=1.5pt]{-2.2}{3.2}{2}
  \psdot[linecolor=blue, linewidth=1.5pt, dotstyle=o](2,2)
\end{pspicture}

\newpage

Existem alguns casos em que o limite pode ser diferente ŕ direita e ŕ esquerda.

\[ \lim _{x \to 0} = \frac{1}{x} \]

\begin{pspicture}(-3,-1.5)(1.5,1.5)
  \psset{xunit=0.5cm,yunit=0.5cm} 
  \psaxes{<->}(0,0)(-3,-3)(3,3)
  \psplot[linecolor=blue, linewidth=1.5pt]{-3}{-0.3}{1 x div}
  \psplot[linecolor=blue, linewidth=1.5pt, arrows=->]{0.3}{3}{1 x div}
\end{pspicture}

Assim,

\[ \lim _{x \to 0^+} \frac{1}{x} = \infty \]
\[ \lim _{x \to 0^-} \frac{1}{x} = -\infty \]

\newpage

Um exemplo mais complexo:

\[ \lim _{x \to 3} \frac{x^2 - 6x + 9}{x^2 - 9} = \frac{0}{0} \]
\[ \lim _{x \to 3} \frac{(x-3)(x-3)}{(x+3)(x-3)} = \frac{x-3}{x+3} = \frac{0}{6} = 0 \]

Quando năo é possível simplificar, pode-se considerar os maiores termos:

\[ \lim _{x \to \infty} = \frac{x^2 + 3}{x^3} = 0 \]

\[ \lim _{x \to \infty} = \frac{3x^2+x}{4x^2-5} = \frac{3}{4} \]

\newpage

Outro exemplo complexo:

\[ \lim _{x \to \infty} \sqrt{x^2 + 4x + 1} - x \]
\[ \lim _{x \to \infty} \sqrt{x^2 + 4x + 1} - x \cdot \frac{\sqrt{x^2 + 4x + 1} + x}{\sqrt{x^2 + 4x + 1} + x} \]
