O eProfile é um gerenciador de perfis voltado para ambientes distribuídos, que infere dados de perfil através da inferência das trilhas de uma entidade. A trilha é disponíbilizada por um gerenciador de trilhas que se conecta ao eProfile, onde é feita uma mediação entre o formato de dados da trilha e do eProfile.

A ontologia de perfil do eProfile pode ser extendida pelos clientes, de modo que os clientes possam armazenar dados de perfil que interessem apenas a si mesmos. Assim, os clientes podem disponibilizar suas próprias regras para a inferência das informações relacionadas à sua própria estrutura de dados. Os dados inferidos ficam então disponíveis para serem consultados pelas entidades.
