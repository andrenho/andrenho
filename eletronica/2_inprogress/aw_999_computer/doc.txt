=========
=  CPU  =
=========

Registers (8-bit):
 8-bit  (A, B, C, D, E, F, G, H)
 32-bit (PC, SP)

Flags:
 Y (carry)
 Z (zero)
 I (interrupt disable)
 O (overflow)
 N (negative)

Types of operations:
 - Load/store (from/to memory)
     LD, ST
 - Register transfers
     SET
 - Stack operations
     PUSH, POP
 - Logical / shift
     AND, XOR, OR, NOT, SHL, SHR, ROL, ROR, TEST
 - Arithmetical / increment
     ADD, SUB, MUL, DIV, INC, DEC
 - Jumps / branches
     JMP, CALL, RET
 - Test / status flags
     BR
 - System functions
     INT, RETI, NOP
